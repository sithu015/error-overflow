# error-overflow
Notes relating to program installation errors, library import errors ...

Linux OS အပေါ်မှာ ပရိုဂရမ်ကို installation လုပ်တဲ့ အခါမှာတွေ့ရတတ်တဲ့ error တွေ၊ ကျွန်တော် နေ့စဉ်လိုလိုသုံးဖြစ်တဲ့ Perl, Python programming နဲ့ ပတ်သက်တဲ့ library import  လုပ်တဲ့အခါမှာတွေ့ရတဲ့ error တွေကို ဖြေရှင်းပုံ ဖြေရှင်းနည်း၊ တခြား အသုံးဝင်တဲ့ tool စတာတွေနဲ့ ပတ်သက်ပြီး ကြုံရင်ကြုံသလို ရေးသွားပါမယ်။ ကျွန်တော်ကိုယ်တိုင်က ကိုယ်ရဲ့ စက်တွေထဲမှာ note.txt ဖိုင်တွေနဲ့ သိမ်းထားတာမျိုးတွေရှိပေမဲ့ သိကြတဲ့အတိုင်းပဲ GitHub မှာတင်ထားတာက ရှာရဖွေရတာ လွယ်ကူတာရေါ၊ မြန်ဆန်တာရော၊ နောက်ပြီးတော့ တစ်ယောက်ယောက်အတွက် အသုံးဝင်ပါလိမ့်မယ်။
