### Solved with following:

sudo apt-get install libasound-dev

pip install portaudio
(I couldn't install ... version error)

Download portaudio from http://portaudio.com/download.html

cd ~/Downloads/
cd portaudio/
./configure
make
sudo make install

pip install pyaudio

Solved!!!
