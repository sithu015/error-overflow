## Preparing a Python Script

နဂို ရှိပြီးသား python code ကို အောက်ပါအတိုင်း update လုပ်ခဲ့...  

```python
"""
XML ဖိုင်ထဲကနေ သတ်မှတ်ထားတဲ့ tag ထဲက content တွေကိုပဲ extraction လုပ်ဖို့ ရေးခဲ့...  
Written by Ye Kyaw Thu, Visiting Professor, LST, NECTEC, Thailand
Last Updated: 7 Nov 2021

How to run: python ./xml2txt.py <folder-name> <xml-tag-name>
e.g. python ./xml2txt.py bibles/ seg
"""

import os
import sys
import xml.etree.ElementTree as ET

# XML ဖိုင်ကနေ argument parsing လုပ်လိုက်တဲ့ XML-tag ရဲ့ အထဲက စာကြောင်းတွေကိုပဲ ဆွဲထုတ်ယူပြီး
# text ဖိုင်အသစ်ထဲကို သိမ်းပေးမယ့် Function
def xml_to_txt(xml_file, xml_tag):
    print('start conversion for ', xml_file, '...')
    root = ET.fromstring(open(xml_file).read())
    with open(xml_file + '.txt', 'w', encoding='utf-8') as out:
        for n in root.iter(xml_tag):
            out.write(n.text.strip() + '\n')


# ဘယ်ဖိုလ်ဒါအောက်က ရှိတဲ့ XML ဖိုင်တွေလည်း ဆိုတာအတွက် folder-name ကို assign လုပ်တာ
directory = sys.argv[1]
# XML tag နာမည်ကို assign လုပ်တာ
xml_tag = sys.argv[2]
 
for filename in os.listdir(directory):
    xml_file = os.path.join(directory, filename)
    # ဖိုင် ဟုတ်ရဲ့လား ဆိုတာကို စစ်တာ
    if os.path.isfile(xml_file):
        # .xml ဆိုတဲ့ file extension နဲ့ ဆုံးတဲ့ ဖိုင်တွေကိုပဲ အလုပ်လုပ်မယ်
        if xml_file.endswith('.xml'):        
            xml_to_txt(xml_file, xml_tag)

            
```

"""
